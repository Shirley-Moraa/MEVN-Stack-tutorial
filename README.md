# vue-mevn-stack-app
git clone the project

## Project setup
```
cd vue-mevn-stack-app
npm install
```

### Compiles and hot-reloads for development
```
Create a terminal window for MongoDB
run: mongo

cd backend

start node server: http://localhost:4000/api
nodemon

start vue app: http://localhost:8000
npm run serve
```
